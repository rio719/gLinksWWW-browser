How to use gLinksWWW 🇺🇸

  • Multi-Copy&Cut&Paste(18 slot multi clipboard): Ctrl+Shift+ 0 ~ 9/f1 ~ f8 (total 18 slots) for copy,Alt + Shift + 0 ~ 9/f1 ~ f8 (total 18 slots) for cut, Ctrl+Alt+ 0 ~ 9/f1 ~ f8 (total 18 slot) for paste. (Example: Ctrl+Shift+1 to copy, Ctrl+Alt+1 to paste || Alt+Shift+f2 to cut, Ctrl+Alt+f2 to paste)
  • Cookie Vault: Click the cookie icon to destroy cookies.
  • manage clipboard: click the paper icon to view your clipboard contents and manage them.
  • Search: choose the search engine, type anything in the search bar and click the 🔍 button.
  • Travelling the Web: Type any link in the top bar and click the 'Go!'button
  • Tabs: Click the + button to open a new tab.
  • Home Button: Click the Home button to return to the home page.
  • Navigation: Use the back and forward buttons to navigate through your browsing history.
  • Reload: Click the reload button to refresh the current page.
  • Watch youtube video/shorts: Click a youtube video/shorts and click the reload button

지링스www 사용법 🇰🇷

  • 다중 복붙 & 자르기 (18슬롯 다중클립보드): Ctrl+Shift+ 0 ~ 9/f1 ~ f8 로 복사하고, Alt + Shift + 0 ~ 9/f1 ~ f8 로 자르기, Ctrl+Alt+ 0 ~ 9/f1 ~ f8 로 붙여놓기. (Example: Ctrl+Shift+1로 복사한 뒤, Ctrl+Alt+1로 붙여놓기 || Alt+Shift+f2로 자른 뒤, Ctrl+Alt+f2로 붙여놓기)
  • 쿠키 관리: 쿠키 버튼을 눌러 웹사이트별로 쿠키를 관리해보세요!

  • 검색: 검색 엔진을 선택하고, 검색창에 내용을 입력한 뒤 🔍 버튼을 클릭하세요.
  • 웹 탐험: 주소창에 주소를 입력한뒤, 'Go!' 버튼을 클릭하세요.
  • 탭: + 버튼을 눌러 새 탭을 열 수 있습니다.
  • 홈버튼: 홈 버튼을 눌러 홈 화면으로 돌아갈 수 있습니다.

  • 새로고침: 새로고침 버튼을 눌러 현재 페이지를 새로 고칠 수 있습니다.

cara memakai gLinksWWW 🇮🇩

  • Multi copy/cut dan paste (18 slot multi clipboard): Ctrl+Shift+ 0 ~ 9/f1 ~ f8 untuk copy, Alt + Shift + 0 ~ 9/f1 ~ f8 untuk cut, Ctrl+Alt+ 0 ~ 9/f1 ~ f8 untuk paste. (Contoh: Copy dengan Ctrl+Shift+1, Paste dengan Ctrl+Alt+1 || Cut dengan Alt+Shift+f2, Paste dengan Ctrl+Alt+f2)
  • Mengatur cookie: Anda bisa mengatur cookie per website dengan mengeklik tombol cookie.
  • Mengelola clipboard: Klik ikon kertas untuk melihat isi clipboard Anda dan mengelolanya.
  • Search: Pilih mesin pencari, masukkan teks di kolom pencarian, lalu klik tombol 🔍.
  • Web browsing: Masukkan alamat di kolom alamat dan klik tombol 'Go!'.
  • Tab: Klik tombol + untuk membuka tab baru.
  • Home button: Klik tombol Home untuk kembali ke halaman utama.
  • Navigasi (back & forward): Gunakan tombol back dan forward untuk kembali ke halaman sebelumnya.
  • Reload: Klik tombol reload untuk memuat ulang halaman saat ini.
  • Nonton youtube video/shorts: Klik youtube video/shorts yang anda ingin menonton. Setelah itu, klik tombol reload.
